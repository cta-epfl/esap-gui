# ESAP-GUI

ESCAPE ESFRI Science Analysis Platform - GUI

This frontend (ReactJS) connects to the ESAP-api-gateway (Django).

## Documentation

Local deployment

```
npm install
npm start
```
or
```
yarn install
yarn start
```
